# “宠小镜”（原“镜小宠”）宠物照料 AI 方案

## 项目概述

### 项目名称
宠小镜 —— 由“镜小宠”正式更名，面向乐奇眼镜、手机端与智能体协同的宠物照料助手

### 核心定位

依托乐奇智能眼镜语音交互、免手操作优势，结合手机端沉浸式记录与官网/后端数据服务，聚焦宠主核心需求，打造 “记录 + 提醒 + 基础诊断” 轻量型宠物照料系统，让基础宠物照料更便捷、更规范。

### 三位一体架构

1. **智能体（宠小镜）**：运行于乐奇眼镜，负责实时语音交互、指令执行与即时提醒。
2. **手机端 APP**：承担数据补录、可视化统计、消息推送等扩展体验，实现随时随地的管理能力。
3. **眼镜端体验**：通过免手操作与场景化 UI，实现户外、遛宠等场景的快速记录。
4. **官网/后端**：统一提供账号体系、数据同步、开放 API 以及后台运营配置，支撑全端一致体验。

四个子系统共享统一数据库与服务接口，确保场景信息流转顺畅。

## 目标用户与核心痛点

### 目标用户
25-40 岁都市养狗人群（新手宠主、上班族、规律养宠用户）。

### 核心痛点
1. 手动记录繁琐，易遗漏喂食、遛宠、排泄情况；
2. 无科学提醒，喂食 / 遛宠时间凭记忆，易打乱宠物作息；
3. 新手难通过排泄、饮食状态判断宠物健康。

### 核心功能模块（全端协同）
1. 语音快速记录
    1. 喂食记录：语音指令 “镜小宠，记录喂食”，自动录入时间、默认喂食量（可语音补充 “50g 狗粮”）；
    2. 遛宠记录：语音说 “镜小宠，记录遛宠”，同步眼镜定位，记录遛宠起始时间；
    3. 排泄记录：语音指令 “镜小宠，记录排便”“镜小宠，记录排尿”，自动标注时间与状态（正常 / 异常，异常可语音补充 “软便”）。
2. 智能定时提醒
    1.宠主预设喂食、遛宠时段（如 “早 7 点喂食”“晚 8 点遛宠”），到点语音提醒 “该给毛球喂食啦”；
    2. 异常提醒：若超过 24 小时未记录排泄 / 遛宠，智能体主动提示 “毛球已超 1 天未排便，请注意观察”。
3. 基础健康诊断
    1. 语音咨询：宠主说 “镜小宠，它一天没排便”，智能体结合知识库，语音反馈基础建议 “可能饮水不足，建议增加饮水量，2 天无改善需就医”；
    2. 数据关联诊断：基于记录的饮食、排泄数据，每周生成简短健康提示 “本周喂食规律，排泄正常，继续保持～”。

## 总结
“宠小镜” 以 “轻量、实用、易落地” 为核心，精准解决宠主基础照料痛点，并通过“智能体 + 手机端 + 眼镜端 + 官网/后端”三位一体架构承接未来迭代。项目开发成本低、演示效果直观，为后续功能扩展预留空间，具备较强竞争力。
